# Rules.md - Minecraft Block Trivia Game Rules

## Game Objective
Test your Minecraft block knowledge by answering trivia questions about block properties, generation, crafting, and special characteristics.

## Game Flow

### 1. Start Screen
- **Display**: Title "MINECRAFT BLOCK TRIVIA" with hero image and game features listed
- **Features Highlighted**:
  - 5 questions per game
  - Multiple choice answers
  - Test your block knowledge
- **Action**: Click "START GAME" button to begin

### 2. Question Screen (5 questions total)
Each question displays:
- **Question Number**: "Question X of 5" with progress bar showing completion
- **Block Information**:
  - Block texture image (from minecraft.wiki)
  - Block name label
- **Question Text**: The trivia question about the block's properties
- **Answer Options**: 5 clickable buttons with randomized answer choices
  - Labeled A, B, C, D, E
  - Correct answer position is randomized each game
  - Answers are plausible distractors generated from other blocks' data
- **Badges**: Category (mechanical, generation, crafting, etc.) and difficulty (easy, medium, hard)

### 3. Answering a Question
1. Click one of the 5 answer buttons to select your answer
2. The selected button highlights in blue
3. Click "Submit Answer" button to confirm
4. Immediate feedback is shown

### 4. Feedback Display
After submitting an answer:
- **Correct Answer**: Shown in green with "✓ Correct!" message
- **Incorrect Answer**: Shown in red with "✗ Incorrect" message
- **Explanation**: Contextual feedback combining the block name with key property information
- **Visual Feedback**: Animations and sound effects play
- **Next Button**: "Next Question" button appears to advance

### 5. Score Tracking
- **Live Score**: Displayed at the top of the game screen as "X / 5"
- **Accuracy Percentage**: Shows current accuracy percentage
- **Score Updates**: Increments by 1 for each correct answer
- **No Penalties**: Incorrect answers do not reduce score, only correct answers count

### 6. Game Over Screen
After all 5 questions:
- **Final Score**: Displayed as percentage (0%, 20%, 40%, 60%, 80%, 100%)
- **Score Breakdown**: Shows correct vs. incorrect count
  - ✓ Correct: X
  - ✗ Incorrect: Y
- **Performance Message**: Contextual message based on score:
  - 100%: "🎉 Perfect! You are a Minecraft master!"
  - 80-99%: "🎮 Great job! You know your blocks!"
  - 60-79%: "👍 Good effort! Keep playing!"
  - Below 60%: "📚 Try again to improve your knowledge!"
- **Hero Image**: Game over celebration image displayed
- **Play Again**: Green button to restart the game and return to Start Screen

## Question Generation

### Trivia Hooks
Each block in the corpus has 3-5 "trivia hooks" - seed questions about different aspects:
- **Mechanical**: Mining times, tool requirements, hardness, blast resistance
- **Generation**: Spawn locations, Y-levels, biomes, rarity
- **Crafting**: Recipes, yield amounts, Fortune compatibility
- **Special**: Luminosity, transparency, flammability, physics effects
- **Variants**: Color types, oxidation states, block variations

### Incorrect Answers (Distractors)
For each question, 4 incorrect answers are generated by:
1. Extracting property values from OTHER blocks in the corpus
2. Mining similar or related answers (e.g., other tool names, Y-levels)
3. Filtering out answers that are too obviously wrong
4. Ensuring plausibility to make the question challenging

Example:
- **Question**: "What is the minimum tool tier to mine Diamond Ore?"
- **Correct**: "Iron Pickaxe (or better)"
- **Distractors**: "Stone Pickaxe", "Wooden Pickaxe", "Shovel", "No tool required"

## Scoring & Progression

- **One Game**: 5 consecutive questions
- **Score Calculation**: Correct answers only (no negative scoring)
- **Accuracy**: Percentage shown in real-time and at end
- **Progression**: Linear - must answer all 5 questions in sequence
- **No Branching**: No skip button, no difficulty selection mid-game

## Block Coverage

### Current Corpus (25 blocks)
**Ores** (4): Diamond, Iron, Gold, Copper
**Building Materials** (5): Stone, Oak Wood, Birch Wood, Blue Concrete, Bricks
**Decorative** (3): Glowstone, Sea Lantern, Lantern
**Functional/Mechanical** (5): Sculk, Redstone Dust, Piston, Hopper, Observer
**Natural** (3): Grass Block, Sand, Gravel, Obsidian, Ice
**Special** (3): Amethyst Block, Bedrock, End Portal Frame

### Categories Covered
- Mechanical properties (hardness, tool tiers, mining times)
- Generation (Y-levels, biomes, spawn rates)
- Crafting (recipes, yields, Fortune)
- Special properties (luminosity, transparency, effects)
- Variants (colors, oxidation, block states)

## User Experience Details

### Audio
- Click sound when selecting an answer
- Correct answer sound (ascending tones)
- Incorrect answer sound (descending buzz)
- Game Over fanfare (victory sequence)
- Next question sound (quick beep)

### Animations
- Fade-in: Hero images, text, feature boxes
- Slide-up: Main containers, cards
- Bounce: Titles ("MINECRAFT BLOCK TRIVIA")
- Pulse: Score percentage and updates
- Shake: Incorrect answer feedback (if included)

### Responsiveness
- Mobile (375px): Full-width layout, stacked buttons
- Tablet (768px): Adjusted spacing, readable text
- Desktop (1280px+): Optimal spacing, button arrangements

## Accessibility
- All buttons have clear labels and click states
- Text contrast meets accessibility standards
- Keyboard support (Enter to advance past feedback)
- Color + symbols used for feedback (not color-only)
- Reduced motion: Custom animation classes respect prefers-reduced-motion
